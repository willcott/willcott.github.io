About the Game of Life

The Game of Life is set in an infinite two-dimensional grid inhabited by “cells”.
Each cell interacts with its neighbours to determine the next state of the grid.

If a living cell has less than two neighbours, there is underpopulation, so the cell dies.

If a living cell has has more than three neighbours, there is overcrowding, so the cell dies.

A cell is born where there is an empty position with three neighbours.

How it was Implemented

This implementation was achieved by storing the coordinates of live cells in an array.

To compute the next iteration, the number of neighbours of each living cell is checked.

If the number of neighbours is not two or three, then the live cell is removed from the array.

The only way a dead cell can come to life is if it has neighbouring live cells.

Therefore each neighbour of the live nodes has its number of neighbours checked, if it is three, it is added to the array.
